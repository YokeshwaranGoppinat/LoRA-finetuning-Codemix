# LoRA Fine-Tuning Project (Admissions Ready)

This repository contains a clean, admissions-ready implementation of LoRA fine-tuning using HuggingFace PEFT.

## Structure
- `notebooks/` — cleaned LoRA notebooks
- `run_colab.ipynb` — one-click runnable Colab pipeline
- `requirements.txt` — pinned dependencies
- `LICENSE` — MIT License
- `.gitignore` — ignore cache/checkpoints

## How to Run (Colab)
1. Upload the repo to Colab.
2. Run `run_colab.ipynb` to install dependencies and execute smoke tests.

## Notes
- Uses HuggingFace PEFT for LoRA.
